Line Graphs and Scatter Plots

Make sure you link the d3 script, the dataviz, and jquery script: 
  <script src="d3.min.js"></script>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.0/jquery.min.js"></script> 
  <script src="datavis.js"></script>
Data Structure for function input: The line graph and the scatter plot have a similar standard data structure. Both need a specified header dictionary, and data dictionary. The absolutely necessary elements is the entire header dictionary and the content block inside the data dictionary. The "content" should contain an array of dictionaries formatted in the example below.
The x axis can either label numbers or Dates. In order to use the Date labelling, the dictionary graph_ticks MUST be defined with its corresponding elements to define how the ticks should be marked. Example of Line Graph with Dates: 
chartinfo = {
  header:{
    title:{
      text: "Line Graph with Dates"
    },
    id:"scattery" // graph id, if you have multiple graphs, make sure you have this param
  },
  data:{
    // content and colors correspond data in dictionary at index 1 will be colored in the color at index 1 <- if data exceeds colors, colors will loop via modular arithmetic
    content: [
                {"Title":"Reservations By Date","Items":[{"Key":"1/21/2010 12:00:00 AM","Value":124},{"Key":"1/22/2010 12:00:00 AM","Value":5},{"Key":"1/24/2010 12:00:00 AM","Value":1},{"Key":"1/25/2010 12:00:00 AM","Value":37},{"Key":"1/26/2010 12:00:00 AM","Value":42},{"Key":"1/27/2010 12:00:00 AM","Value":42},{"Key":"1/28/2010 12:00:00 AM","Value":32},{"Key":"1/29/2010 12:00:00 AM","Value":19},{"Key":"1/30/2010 12:00:00 AM","Value":10},{"Key":"1/31/2010 12:00:00 AM","Value":17},{"Key":"2/1/2010 12:00:00 AM","Value":44},{"Key":"2/2/2010 12:00:00 AM","Value":36},{"Key":"2/3/2010 12:00:00 AM","Value":42},{"Key":"2/4/2010 12:00:00 AM","Value":39},{"Key":"2/5/2010 12:00:00 AM","Value":26},{"Key":"2/6/2010 12:00:00 AM","Value":11},{"Key":"2/7/2010 12:00:00 AM","Value":21},{"Key":"2/8/2010 12:00:00 AM","Value":52},{"Key":"2/9/2010 12:00:00 AM","Value":48},{"Key":"2/10/2010 12:00:00 AM","Value":3},{"Key":"2/11/2010 12:00:00 AM","Value":36},{"Key":"2/12/2010 12:00:00 AM","Value":27},{"Key":"2/13/2010 12:00:00 AM","Value":14},{"Key":"2/14/2010 12:00:00 AM","Value":22},{"Key":"2/15/2010 12:00:00 AM","Value":56},{"Key":"2/16/2010 12:00:00 AM","Value":53},{"Key":"2/17/2010 12:00:00 AM","Value":67},{"Key":"2/18/2010 12:00:00 AM","Value":58},{"Key":"2/19/2010 12:00:00 AM","Value":33},{"Key":"2/20/2010 12:00:00 AM","Value":18},{"Key":"2/21/2010 12:00:00 AM","Value":37},{"Key":"2/22/2010 12:00:00 AM","Value":68},{"Key":"2/23/2010 12:00:00 AM","Value":51}]}

    colors: ["blue", "red", "green"], //dotcolor

   graph_tick:{
      date_by: "day", //, "month", "day"

      label: "%m/%d/%Y"

  }
}
