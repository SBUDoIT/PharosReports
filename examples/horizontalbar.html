Creating Horizontal Bar Graph

Make sure you link the d3 script,and dataviz script: 
  <script src="d3.min.js"></script>
  <script src="datavis_test.js"></script>

JavaScript Function:

To run, call: makeHorizontalBars( arrayOfData, idOFelementToAppendChart, colors);
The idOfelementtoAppendchart is the id without # of the element the chart will be appended to.
The chartinfo should be formatted as follows: (This may be changed in the near future as this may not be the finalized data structure for the input data).
The colors parameter is an OPTIONAL parameter where you can choose colors you want your bars to have. If this is empty, this default array of colors will be used. (Note the bar colors will loop if there are more bars than colors). : 
["c0392b", "8e44ad", "f39c12", "1abc9c", "d35400", "2980b9", "e74c3c", "2c3e50", "2ecc71", "f1c40f", "ecf0f1", "16a085", "7f8c8d"]
#### NOTE: Items Values should be ordered from greatest to least. ####

Data Structure for function input

chartinfo={"Title":"Reservations By Status",
  "Items":[

    {"Key":"Key 2","Value":40},
    {"Key":"Key 3","Value":30},
