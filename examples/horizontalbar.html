Creating Horizontal Bar Graph

Make sure you link the d3 script,and dataviz script: 
  <script src="d3.min.js"></script>
  <script src="datavis_test.js"></script>

JavaScript Function:

To run, call: makeHorizontalBars( arrayOfData, idOFelementToAppendChart);
The idOfelementtoAppendchart is the id without # of the element the chart will be appended to.
The arrayOFData should be formatted as follows: (This may be changed in the near future as this may not be the finalized data structure for the input data).
#### NOTE: Items Values should be ordered from greatest to least. ####

Data Structure for function input

chartinfo={"Title":"Reservations By Status",
  "Items":[
    {"Key":"Key 1","Value":50},
    {"Key":"Key 2","Value":40},
    {"Key":"Key 3","Value":30},
    {"Key":"Key 4","Value":20},
    {"Key":"Key 5","Value":10},
    ]
}
Code Run: makeHorizontalBars( arrayOfData, idOFelementToAppendChart);

Please note that this data shown above is not drawn from any source and is completely made-up.
And PLEASE note how ugly a long sentence label will make the graph!
